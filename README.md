# Nauvoo Family Adventure Website

A mobile-first itinerary webpage for the July 23–27, 2026 Nauvoo / St. Louis family trip.

## How to launch on GitHub Pages

1. Create a new GitHub repository.
2. Upload `index.html` and `style.css` to the root of the repository.
3. Go to **Settings → Pages**.
4. Under **Build and deployment**, choose:
   - Source: Deploy from a branch
   - Branch: main
   - Folder: /root
5. Save.
6. GitHub will give you a public URL in a minute or two.

## How to customize

- Replace image URLs in `index.html` with your own photos.
- Update hotel links once you pick the exact hotel.
- Update flight details once booked.
- Add or remove day cards inside the `timeline` section.
